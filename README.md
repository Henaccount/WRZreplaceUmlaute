# WRZreplaceUmlaute
decompresses wrz files, replaces german special characters and compresses back to wrz format.

Replace("ä", "ae");

Replace("ö", "oe");

Replace("ü", "ue");

Replace("Ä", "Ae");

Replace("Ö", "Oe");

Replace("Ü", "Ue");

Replace("ß", "ss");

Usage: compile e.g. in MS Visual Studio, then call the WRZreplaceUmlaute.exe from the bin folder.
